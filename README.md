# 🤖 Telegram Media Downloader Bot

بوت تيليجرام لتحميل الفيديوهات والصوتيات من مختلف المنصات.

## ✨ المميزات

- ✅ دعم أكثر من 1000 موقع (YouTube, Instagram, TikTok, Twitter, Facebook...)
- 🎵 تحميل الصوت بصيغة MP3
- 🎬 تحميل الفيديوهات بجودة حتى 720p
- 🔄 عرض نسبة التقدم أثناء التحميل
- 🗑️ تنظيف تلقائي للملفات القديمة
- 🌐 يعمل على Render بشكل دائم

## 📋 المتطلبات

- Python 3.11+
- حساب على Telegram
- حساب على Render (مجاني)
